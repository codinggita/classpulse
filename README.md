## 📊 ClassPulse

**Real-Time Classroom Polling Extension**

ClassPulse is a browser extension that allows teachers to run quick polls during lectures to check whether students understand a topic. Students can respond instantly, and teachers can view the results in real time.

## Problem

Teachers often find it difficult to know if students truly understand the concepts being taught. Many students hesitate to ask questions, which can lead to confusion and learning gaps.

## Solution

ClassPulse allows teachers to launch instant polls such as **“Did you understand this topic?”**. Students answer through the extension, and the teacher receives live results to better adjust the lecture.

## Key Features

* Instant classroom polls
* Real-time student responses
* Live result view for teachers
* One response per student
* Poll history *(future improvement)*
